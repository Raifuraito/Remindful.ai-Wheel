# Reward Arcade

A GitHub/Vercel-ready Next.js app based on the full idea list: Ultra Wheel, Horse Plinko, floating collectibles, pets, daily streaks, task rewards, Minesweeper progression, avatar equipment, and an admin tuning panel.

Run locally:

npm install
npm run dev

Open http://localhost:3000.

Put it on GitHub:

git init
git add .
git commit -m "Initial Reward Arcade app"
git branch -M main
git remote add origin https://github.com/YOUR-NAME/YOUR-REPO.git
git push -u origin main

Deploy on Vercel by importing the GitHub repo and keeping the default Next.js settings. This version uses browser localStorage, so it does not need a database yet.
